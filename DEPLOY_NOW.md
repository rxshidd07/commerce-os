# 🚀 DEPLOY TO VERCEL - QUICK GUIDE

Your GitHub repository is ready! Now deploy to Vercel in 2 minutes.

## ⚡ Fast Deployment Steps

### Step 1: Go to Vercel
Click here: **https://vercel.com/new**

### Step 2: Login with GitHub
- Click **"Continue with GitHub"**
- Authorize Vercel to access your GitHub account

### Step 3: Import Your Repository
- Click **"Import Git Repository"**
- Paste this URL:
  ```
  https://github.com/rxshidd07/commerce-os
  ```
- Click **"Continue"**

### Step 4: Configure & Deploy
- Framework: `Next.js` (auto-detected)
- Build Command: `npm run build` (auto-filled)
- Environment Variables: Leave blank for now
- Click **"Deploy"**

### Step 5: Wait for Deployment ⏳
You'll see build logs. Once complete (2-3 minutes), you get a live URL:

```
✅ Deployment successful!
🎉 Your project is live at: https://commerce-os-xxxxx.vercel.app
```

## 🆘 Having Issues?

**Issue:** "Repository not found"
- **Solution:** Make sure your GitHub repo is public
- Check: https://github.com/rxshidd07/commerce-os

**Issue:** Build fails on Vercel
- Check build logs in Vercel dashboard
- Usually just a missing dependency
- Click "Redeploy" to retry

**Issue:** Need custom domain
- Vercel → Settings → Domains
- Add your domain (e.g., commerce-os.io)
- Update DNS records at your domain provider
